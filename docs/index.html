🖼️ WallGen — Multi-Model Wallpaper Generator

A lean, self-hosted text-to-image pipeline that picks the most optimized model + backend for your hardware — MLX on Apple Silicon, Diffusers + GGUF on CUDA/CPU — quantizes transformer and text encoder for a tiny footprint, and stretches the result to your screen.

Z-Image-Turbo · FLUX.2-klein · sd-turbo MLX + Diffusers backends Apple Silicon / CUDA / CPU auto <8GB quantized LoRA-ready No telemetry

What it does

Best model per host

A priority registry + memory probe pick the best model that fits: Z-Image-Turbo → FLUX.2-klein → sd-turbo. VRAM and system RAM are weighed separately on CUDA.

Optimal backend per OS

Apple Silicon runs MLX (mflux) for zero-copy unified-memory speed; CUDA/CPU run Diffusers + GGUF with the right dtype (bf16/fp16/fp32) and offload tier.

Tiny footprint

Transformer and text encoder are quantized — Z-Image-Turbo 4-bit on Apple Silicon is a ~6GB download and runs under 8GB.

Lean output + prompt lock

No upscaler: generate native (≤1080p) then stretch-resize. A wallpaper directive tells the model it'll be stretched; a prompt lock keeps the deployed scheduler on-rails.

Design decisions

MLX on Apple Silicon

On unified memory, CPU offload only duplicates weights. MLX (mflux) is zero-copy and far faster than PyTorch-MPS — so it's the Apple-Silicon backend, loading resident.

VRAM ≠ RAM on CUDA

Selection weighs transformer-on-VRAM separately from total-on-RAM (encoder offloads to RAM). dtype follows the GPU: bf16 on Ampere+, fp16 on Turing.

Quantize the encoder too

The text encoder (Qwen3-4B) is the real memory hog. Quantizing it — not just the transformer — is what gets the whole model under 8GB.

Stretch, don't upscale

Generate at a native ≤1080p size and stretch-resize — no upscaler dependency. The prompt directive tells the model it'll be stretched so it frames accordingly.

Quickstart

uv sync --extra mlx --extra ui --extra lora          # Apple Silicon (MLX)
# uv sync --no-group cpu --group cuda --extra ui      # NVIDIA GPU
# uv sync                                             # no GPU (lean CPU torch)

uv run python -m wallgen generate "northern lights over a fjord"
uv run python -m wallgen batch prompts/wallpapers.txt
uv run python -m wallgen ui                          # localhost Gradio UI
uv run python -m wallgen generate "neon city" --lora some-org/neon:0.8
